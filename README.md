# FilesSpy
一个可以静默偷取文件的Python脚本

## 运行方法

在Python环境下，运行**RunFilesSpy.py**文件(其实直接运行**FilesSpy.py**也行，只是没有静默模式)。

**RunFilesSpy_PyInstaller.py**适用于封装Windows的可执行文件，需要将Python解释器封装到exe之中，可以在**Releases**中下载封装后的文件
